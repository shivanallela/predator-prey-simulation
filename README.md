This project simulates a classic predator-prey ecosystem using the Mesa agent-based modeling framework in Python. 

- Predators (red) hunt prey (green) on a grid.
- Users can interactively run the simulation, step through time, and observe population dynamics.
- Parameters like reproduction rate, initial population, and predator behavior can be adjusted.

Built with Python 3.14 and Mesa, this project is ideal for learning agent-based modeling and ecological simulations.# predator-prey-simulation
